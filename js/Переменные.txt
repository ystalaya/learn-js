Переменная и константа,
идентификатор,
декларация переменных,
инициализация переменных,
литерал

Переменная – это «именованное хранилище» для данных. Мы можем использовать переменные для хранения товаров, посетителей и других данных.
Для создания переменной в JavaScript используйте ключевое слово let.

Переменные, объявленные с помощью const, называются «константами». Их нельзя изменить. Попытка сделать это приведёт к ошибке

идентификатор - уникальное имя переменной, функции, массива. 

декларация переменных происходит через let/var/const 

инициализация переменных - это присвоение значения переменным

Литерал – это любое значение указанное явным образом в коде. В качестве литералов в js могут выступать числа, строки (текстовые значения), логические значения.
